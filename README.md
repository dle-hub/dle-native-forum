# 🚀 DLE Native Forum Ultimate Core

DataLife Engine (DLE) için geliştirilmiş, tamamen native, OOP yapılı ve AJAX tabanlı profesyonel forum eklentisi.

## ✨ Öne Çıkan Özellikler
- **Tam Entegrasyon:** DLE profil sistemi (`userinfo.tpl`) ile tam uyumlu.
- **Kullanıcı Kariyeri:** Profil sayfalarında mesaj sayısı, puan ve rütbe gösterimi.
- **AJAX Altyapısı:** Sayfa yenilemeden beğeni, dislike ve hızlı işlem desteği.
- **Gelişmiş Moderasyon:** Admin panelinde renkli loglama (Move/Bump) ve tam kontrol.
- **SEO Dostu:** Arama motorları için optimize edilmiş URL yapısı.

## 📦 Kurulum
1. `upload` klasöründeki dosyaları FTP ile ana dizine atın.
2. `plugin` klasöründeki güncel XML dosyasını (v1.1.2) DLE Admin -> Eklentiler kısmından yükleyin.
3. Temanızdaki `userinfo.tpl` dosyasına forum istatistik kartını eklemeyi unutmayın!
